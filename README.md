## 🌟 마비노기 경매장 탐정: 당신의 득템을 위한 비밀 병기! 🌟

### 1. 프로덕트 한 줄 소개
이 앱은 마비노기 경매장 데이터를 분석하여, 특정 지역에서 획득 가능한 아이템의 예상 수익률을 Discord로 알려주는 똑똑한 득템 가이드 봇입니다!

### 2. 프로덕트 개요
마비노기에서 어떤 아이템을 파밍해야 돈을 벌 수 있을지 고민이 많으셨죠? '마비노기 경매장 탐정'은 바로 이런 밀레시안들의 고민을 해결하기 위해 태어났습니다! 이 봇은 마비노기 경매장 데이터를 꼼꼼히 분석하여, 특정 사냥터나 채집지에서 얻을 수 있는 아이템들의 현재 시세와 시간당 예상 수익률을 계산해줍니다. 그리고 이 유용한 정보를 Discord 채널로 주기적으로 알려주어, 당신이 언제 어디서든 현명한 파밍 계획을 세울 수 있도록 돕습니다. 이제 더 이상 경매장에서 헤매지 마세요! 당신의 소중한 시간과 골드를 지켜줄 똑똑한 가이드가 여기 있습니다.

### 3. 주요 기능 (사용자 관점)

*   **자동화된 득템 정보 알림: "나만의 똑똑한 정보원"**
    *   이 봇은 15분마다 마비노기 경매장 데이터를 자동으로 수집하고 분석하여, 특정 지역에서 획득 가능한 아이템들의 예상 수익률 정보를 Discord 채널에 자동으로 전송합니다. 마치 나만의 정보원이 쉴 새 없이 꿀 정보를 물어다 주는 것 같죠!
*   **수익성 높은 아이템 추천: "숨겨진 보물을 찾아주는 보물지도"**
    *   분석된 데이터 중 특히 수익성이 높다고 판단되는 아이템에는 🔥 이모지를 붙여 강조합니다. 어떤 아이템을 파밍해야 할지 막막할 때, 이 보물지도가 당신을 득템의 길로 안내할 거예요!
*   **친근하고 명확한 정보 제공: "이해하기 쉬운 경제 뉴스"**
    *   복잡한 경매장 데이터를 '아이템', '단위 시간', '기대 획득 수', '경매장 최저가 평균', '매물 수', '1시간 기대 수익률'과 같은 명확한 표 형식으로 정리하여 제공합니다. 마치 경제 뉴스를 보듯 쉽고 빠르게 정보를 파악할 수 있습니다.

### 4. 사용자 경험 흐름

1.  **Discord 채널 설정**: 봇이 설치된 Discord 서버의 특정 채널에서 봇이 보내는 메시지를 확인합니다. (아마도 관리자가 봇을 서버에 추가하고 알림 채널을 설정할 것입니다.)
2.  **자동 알림 수신**: 15분마다 봇이 자동으로 전송하는 '📢 검쨩봇의 추천노기' 메시지를 확인합니다.
3.  **득템 계획 수립**: 메시지에 포함된 표와 추천 아이템(🔥 이모지) 정보를 바탕으로 "아, 이 지역에서 이 아이템을 파밍하면 돈이 되겠네!"와 같은 현명한 파밍 결정을 내립니다.
4.  **성공적인 파밍**: '마비노기 경매장 탐정'의 도움으로 성공적인 파밍을 하고, 즐거운 마비노기 생활을 만끽합니다!

### 5. 주요 특징 및 장점

*   **완전 자동화**: 사용자가 직접 경매장을 검색하거나 데이터를 분석할 필요 없이, 봇이 모든 것을 알아서 처리합니다.
*   **시간 절약**: 15분마다 업데이트되는 정보를 통해, 게임 플레이 시간을 효율적으로 활용하여 득템 기회를 놓치지 않습니다.
*   **수익 극대화**: 데이터 기반의 정확한 수익률 분석으로, 어떤 아이템을 파밍해야 가장 큰 이득을 얻을 수 있는지 알려줍니다.
*   **접근성**: Discord를 통해 언제 어디서든 편리하게 최신 득템 정보를 받아볼 수 있습니다.
